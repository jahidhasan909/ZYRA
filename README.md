# 👕 ZYRA – Modern Fashion Store

ZYRA is a modern and responsive fashion e-commerce frontend built with **Next.js 16**. The project focuses on delivering a clean shopping experience with smooth animations, reusable components, and a modern UI. All product data is managed using local JSON files, making it a frontend-only application without any backend.

## 🌐 Live Demo

**Live Site:** https://zyra-roan.vercel.app/

---

## ✨ Features

- 🛍️ Modern Fashion Store UI
- 📱 Fully Responsive (Mobile, Tablet & Desktop)
- ⚡ Built with Next.js 16 App Router
- 🎨 Clean & Modern User Interface
- 🚀 Smooth Scrolling using **Lenis**
- 🎥 Beautiful Animations with **Framer Motion**
- 🧩 Reusable & Modular Components
- 📂 Local JSON Data (No Backend)
- 🛒 Add Products to Cart
- ➕ Increase / Decrease Product Quantity
- ❌ Remove Products from Cart
- 💾 Cart Data Stored in Local Storage
- 💰 Automatic Total Price Calculation
- 📄 Product Details Page
- 🆕 Browse Trending & New Arrival Products
- 🛍️ Process to Checkout UI
- 🔥 Toast Notifications
- 🎠 Interactive Sliders using Swiper
- ⚙️ Fast Performance & Optimized Components

---

## 🛒 User Journey

- Browse the latest fashion collections.
- Explore trending and new arrival products.
- View complete product details.
- Add products to the shopping cart.
- Increase or decrease product quantity.
- Remove unwanted products from the cart.
- Cart data remains saved using Local Storage.
- Continue through the checkout process.
